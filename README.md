## 🚀 Deployment Instructions

### 1. Environmental Initialization

Clone the repository and install the required dependencies:

```bash
git clone https://github.com/krishnasai89/Samkhya.git
cd Samkhya
npm install
```

### 2. Run Local Diagnostics (Development Mode)

Boot up the local server pipeline:

```Bash
npm run dev
```

Open http://localhost:3000 inside your browser terminal to view the interactive matrix dashboard.

### 3. Production Compilation Build

Compile and verify the structural data files for production deployment:

```Bash
npm run build
npm run start
```
